invinoveritas

Lightning-powered AI reasoning & decision API

Agents and humans can pay with Bitcoin via the Lightning Network and instantly unlock AI responses.

How it works

1. Send a request to the API
2. The API returns a Lightning invoice
3. Pay the invoice
4. Resend request with Authorization header
5. AI response is unlocked
