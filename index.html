Pay-per-insight API. No subscriptions. No accounts. Just Bitcoin via Lightning.

How it works:

1. Send a request to /reason or /decision
2. Receive a Lightning invoice
3. Pay the invoice with any Lightning wallet (Strike, Phoenix, Muun, etc.)
4. Retry the request with Authorization: L402 <token>:<preimage>
5. Get high-quality AI response instantly
