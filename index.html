invinoveritas

Lightning-powered AI reasoning & decision intelligence API

Agents (or humans) can pay with Bitcoin via the Lightning Network and instantly unlock high-quality AI responses.

---

How it works

1. Send a request to /reason or /decision
2. The API returns a Lightning invoice
3. Pay the invoice
4. Retry request with Authorization header → response unlocked
---

Pricing

Reasoning: 500 sats per request
Decision intelligence: 750 sats per request
---

Endpoints

Reasoning 
POST /reason
Body: 
{
  "question": "Explain Bitcoin as a long-term strategy"
}
Response: 
{
  "status": "success",
  "type": "premium_reasoning",
  "answer": "High-quality structured reasoning..."
}
--- Decision Intelligence 
POST /decision
Body: 
{
  "goal": "Grow capital safely",
  "context": "User holds BTC and cash",
  "question": "Should exposure be increased?"
}
Response: 
{
  "status": "success",
  "type": "decision_intelligence",
  "result": {
    "decision": "Increase exposure slightly",
    "confidence": 0.72,
    "reasoning": "Market structure improving while risk remains moderate.",
    "risk_level": "medium"
  }
}
---

Example request (curl)

curl -X POST https://your-api.onrender.com/reason \
  -H "Content-Type: application/json" \
  -d '{"question":"Explain Bitcoin as a long-term strategy"}'

Why this exists

Most AI tools charge per user. invinoveritas charges per insight using Bitcoin and the Lightning Network.

No accounts. No subscriptions through banks. No platform lock-in.
