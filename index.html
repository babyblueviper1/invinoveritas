How it works (L402):

1. Call /reason, /decision, or MCP callTool
2. Receive HTTP 402 with a Lightning invoice
3. Pay the invoice with any Lightning wallet
4. Retry the exact same request with:
Authorization: L402 <payment_hash>:<preimage>
5. Receive the AI response
