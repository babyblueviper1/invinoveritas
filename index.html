invinoveritas

Lightning-powered AI reasoning API for humans and autonomous agents.

Pay with Bitcoin (Lightning Network) and instantly unlock high-quality AI reasoning. No accounts. No subscriptions. No platform lock-in.

---

How it works

1. Send a request to the API
2. The API returns a Lightning invoice (L402)
3. Pay the invoice
4. Send the request again with the Authorization header
5. The AI response unlocks instantly
