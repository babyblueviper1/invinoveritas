invinoveritas

Lightning-powered AI reasoning API

Agents (or humans) pay via Bitcoin Lightning to unlock high-quality AI insights instantly.

---

How it works

1. Send a request to the API
2. API returns a Lightning invoice
3. Pay the invoice
4. Request again with Authorization header
5. AI response is unlocked
---

Dynamic Pricing

- Base price for reasoning: 500 sats
- Base price for decision intelligence: 1000 sats
- Longer questions may increase cost slightly
- Agents (bots/scripts) may pay ~1.2x
---

Endpoints

Premium Reasoning 
POST /reason
Body: 
{
  "question": "Explain Bitcoin as a long-term strategy"
}
Response: 
{
  "status": "success",
  "type": "premium_reasoning",
  "answer": "High-quality structured answer..."
}
--- Decision Intelligence 
POST /decision
Body: 
{
  "goal": "Grow capital safely",
  "context": "User holds BTC and cash",
  "question": "Should exposure be increased?"
}
Response: 
{
  "status": "success",
  "type": "decision_intelligence",
  "result": {
    "decision": "Increase exposure slightly",
    "confidence": 0.72,
    "reasoning": "Market structure improving while risk remains moderate.",
    "risk_level": "medium"
  }
}

Example request (curl)

curl -X POST https://your-api.onrender.com/reason
---

Why this exists

Most AI tools charge per user. invinoveritas charges per insight using Bitcoin and Lightning.

No accounts. No banks. No platform lock-in. Works for humans and autonomous agents.
