invinoveritas

Lightning-powered AI reasoning API

Agents (or humans) can pay with Bitcoin via the Lightning Network and instantly unlock high-quality AI responses.

---

How it works

1. Send a request to the API
2. The API returns a Lightning invoice
3. Pay the invoice
4. The AI response unlocks instantly
---

Pricing

Subscription: 500 sats / 24 hours
Unlimited reasoning requests during the active period.
